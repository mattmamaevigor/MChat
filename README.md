# MChat 🤖
Умный ИИ-помощник прямо в браузере. Один HTML-файл — никаких серверов и установок.
✨ Возможности
💬 Чат с ИИ на базе Llama 3.3 (Groq)
⚡ Быстрые ответы — Groq один из самых быстрых провайдеров
🆓 Полностью бесплатно
📱 Работает на телефоне и планшете
🔑 Ключ хранится только в твоём браузере
🚀 Как запустить
Получи бесплатный API ключ на console.groq.com
Открой index.html в браузере
Введи ключ — и общайся!
🛠 Технологии
Чистый HTML / CSS / JS — без фреймворков
Groq API — бесплатный и быстрый
Модель: llama-3.3-70b-versatile
